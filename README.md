## Overview

This platform delivers a refined shopping experience from product discovery to checkout. Inspired by the visual language of Balenciaga and Loro Piana — elegance through restraint.

<br/>

---

## Features

**Shopping Experience**
- Minimalist, typographic-first UI with a Black & Gold palette
- High-resolution product gallery with interactive product cards
- Slide-out cart drawer for quick item management
- Multi-step guided checkout flow

**Authentication & Admin**
- Secure JWT-based signup and login
- Protected routes with role-based access
- Admin dashboard for inventory and order management

## Getting Started

**Prerequisites:** Python 3.10+, Node.js 18+, PostgreSQL

<br/>

**1. Clone**
```bash
git clone https://github.com/YOUR_USERNAME/luxury-ecommerce.git
cd luxury-ecommerce
```

**2. Backend**
```bash
cd backend
python -m venv venv

# Windows
.\venv\Scripts\activate
# macOS / Linux
source venv/bin/activate

pip install -r requirements.txt
uvicorn main:app --reload
```
> Runs on `http://localhost:8000` · API docs at `/docs`

**3. Frontend**
```bash
cd ../frontend
npm install
npm run dev
```
> Runs on `http://localhost:5173`
